Regression Analysis and Linear Statistics Model

Posted on 2020-01-02 Edited on 2020-01-21 In Learning Views:

NO PROOF HERE

Sherman-Morrison-Woodbury
$(A+UBV)^{-1}=A^{-1}-A^{-1}U(I+BVA^{-1}U)^{-1}BVA^{-1}$

Inverse Matrix
$A_{22.1} =A_{22}-A_{21}A_{11}^{-1}A_{12}$

Generalized inverse matrix
$AXA=A$
The less the rank of $A$, the more $A^-$ exists.

if $A^2 = A$

  • eigenvalue of $A$ : 0 or 1
  • $tr(A) = rk(A)$

$cov(AX, BY) = Acov(X,Y)B^T$

$BA=0 \leftrightarrows BX\ and\ X^TAX\ are\ independent$

$XAX,\ XBX\ and\ X\sim N \leftrightarrows\ AB = 0$

$X \sim N_n(0, \sigma^2),\ \frac{1}{\sigma^2}X^TX \sim \chi(n)$

$if\ X \sim N_n(\mu, \sigma^2),\ A^T = A,\ rk(A) = r$
$then\ \frac{1}{\sigma^2}X^TAX \sim \chi(r,\mu^TA\mu)$

数据缺失模型相关内容请参考韦博成《统计诊断》
广义线性模型相关内容请参考P. McCullagh, John A. Nelder《Generalized Linear》
