描述符

描述符就是具有约束行为的一个对象属性，而它对属性的控制权限是由描述符里的方法来实现的。

A descriptor is an object attribute with “binding behavior”, one whose attribute access has been overridden by methods in the descriptor protocol.

这些方法包括：__set__(), __get__()和 __delete__()。如果这些方法是为了约束某个对象而定义的话，那么它就是描述符。

说人话版本：如果一个新式类定义了__get__(), __set__(), __delete__()方法中的一个或者多个，那么称之为descriptor。

描述符分三种：data descriptor, non-data descriptor和read-only descriptor。第一种就是既有__set__()又有__get__()的，第二种就是只有__get__()的描述符。
最后一种是既有__set__()又有__get__()的，但是__set__()会抛出AttributeError的。

描述符的实例是类的属性。

比较值得注意的是，get, set, delete这三种方法是从对象的字典中get, set or delete属性，如果要操作的并不是属性而是类内的实例方法的话，也不例外。

if obj = Cls(), then

（1）如果“attr”是出现在Cls或其基类的__dict__中， 且attr是data descriptor， 那么调用其__get__方法, 否则
（2）如果“attr”出现在obj的__dict__中， 那么直接返回 obj.__dict__[‘attr’]， 否则
（3）如果“attr”出现在Cls或其基类的__dict__中

  • （3.1）如果attr是non-data descriptor，那么调用其__get__方法， 否则
  • （3.2）返回 __dict__[‘attr’]

（4）如果Cls有__getattr__方法，调用__getattr__方法，否则
（5）抛出AttributeError
