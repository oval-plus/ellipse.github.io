Classmethod and Staticmethod

Posted on 2020-02-18 Edited on 2020-02-19 In Toys Views:

特新手的问题，写在博客里我都觉得自个儿不好意思，但不会就是不会，没办法。

直接举例子吧，输出的结果是11 9 2012和19 2 2020： 

class Date(object):

    def __init__(self, day=0, month=0, year=0):
        self.day = day
        self.month = month
        self.year = year

    @classmethod
    def from_string(cls, date_as_string):
        day, month, year = map(int, date_as_string.split('-'))
        date1 = cls(day, month, year)
        return date1

    def start(self):
        print(self.day, self.month, self.year)

date2 = Date.from_string('11-09-2012')
date2.start()
Date(19, 2, 2020).start()

@classmethod为类定义提供了可替代构造方法的方法。那么换句话说，以后要改类的功能时，不用改构造函数__init__部分，直接上新对象加@classmethod就行了，维护起来比较简单。
被classmethod包装的对象只是借用了类的方法而已，再换句话说就是这个类里的实例方法直接变成了类方法，而且构造函数直接变成了由这个对象自己定义的。换句话说这对象只在类内运行，不在实例内运行。
写的直白一点，其实就是这样子的： 

def ifrom_string(cls, date_as_string):
    day, month, year = map(int, date_as_string.split('-'))
    date1 = cls(day, month, year)
    return date1
class Date(object):
    def __init__(self, day=0, month=0, year=0):
        self.day = day
        self.month = month
        self.year = year

    def start(self):
        print(self.day, self.month, self.year)

date2 = ifrom_string(Date, '11-09-2012')
date2.start()
Date(19, 2, 2020).start()

输出结果和上面是一样的11 9 2012和19 2 2020，其实就是个装饰器……

然后讲@staticmethod静态方法。
静态方法就是又不用用到实例方法，又不用用到类方法，它就相当于是在类方法内的一个孤立的实例方法。好处就是：

  • 代码易读性.静态方法并不需要self参数
  • 节约内存.不用每个实例都实例化方法

staticmethod主要用途是限定namespace，也就是说这个函数虽然是个普通的function，但是它只有这个class会用到，不适合作为module level的function。这时候就把它作为staticmethod。如果不考虑namespace的问题的话直接在module里面def function就行了。

还是日期的例子，但是增添了个确保输入日期无误的函数。

class Date(object):
    def __init__(self, date_as_string):
        self.date = date_as_string

    @staticmethod
    def is_date_valid(date_as_string):
        day, month, year = map(int, date_as_string.split('-'))
        if day <= 31 and month <= 12 and year <= 3999:
            return day, month, year
        else:
            return 'Invalid'

    def start(self):
        # print(self.date)
        print(self.is_date_valid(self.date))

Date('32-09-2012').start()

就会输出Invalid，如果是输入’11-09-2012’，则会输出：(11, 9, 2012)

最后一个关于@classmethod和@staticmethod的区别的例子，结果是12.56： 

class Pizza(object):
    def __init__(self, radius, height):
        self.radius = radius
        self.height = height

    @staticmethod # 计算pizza面积
    def compute_area(radius):
        return 3.14 * (radius ** 2)

    @classmethod # 计算体积，可以看作一个类
    def compute_volume(cls, height, radius):
        return height * cls.compute_area(radius)

    def get_volume(self):
        return self.compute_volume(self.height, self.radius)

print(Pizza(2,1).get_volume())
